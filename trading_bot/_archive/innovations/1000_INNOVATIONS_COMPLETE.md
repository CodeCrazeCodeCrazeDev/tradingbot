## Usage

### Quick Start

```python
from trading_bot.innovations import quick_start_innovations

# Initialize all 1000 innovations
orchestrator = await quick_start_innovations()

# Process market data
market_data = {
    'prices': price_array,
    'volume': volume_array,
    'sentiment': 0.5
}

# Run full analysis
results = await orchestrator.run_full_analysis(market_data)

print(f"Signal: {results['aggregate_signal']}")
print(f"Confidence: {results['confidence']}")
print(f"Recommendation: {results['recommendation']}")
```

### Using Specific Categories

```python
from trading_bot.innovations import InnovationOrchestrator

orchestrator = InnovationOrchestrator()

# Activate only specific categories
orchestrator.activate_category('QUANTUM_CONSCIOUSNESS', weight=2.0)
orchestrator.activate_category('PSYCHOLOGICAL_WARFARE', weight=1.5)
orchestrator.deactivate_category('COSMIC_TRADING')

# Process with custom weights
results = await orchestrator.process_market_data(market_data)
```
