## 🚀 Quick Start

### Option 1: Windows Launcher
```batch
RUN_ULTIMATE_INTEGRATION.bat
```

### Option 2: Python CLI
```bash
# Paper trading (safe)
python run_ultimate_integration.py --mode paper

# With custom symbols
python run_ultimate_integration.py --mode paper --symbols BTCUSDT,ETHUSDT,EURUSD

# Demo mode
python run_ultimate_integration.py --demo

# Show status
python run_ultimate_integration.py --status

# List modules
python run_ultimate_integration.py --list-modules
```

### Option 3: Python API
```python
from trading_bot.ultimate_integration import (
    UltimateIntegration,
    IntegrationConfig,
    IntegrationMode,
    quick_start
)

# Quick start
system = await quick_start({
    'mode': IntegrationMode.PAPER,
    'symbols': ['BTCUSDT', 'EURUSD'],
    'initial_capital': 100000.0
})

# Start trading
await system.start()

# Get status
status = system.get_status()
print(f"Health: {status.health.value}")
print(f"Active Modules: {status.active_modules}/{status.total_modules}")

# Stop
await system.stop()
```

## ⚙️ Configuration

### IntegrationConfig Options

```python
@dataclass
class IntegrationConfig:
    # Mode
    mode: IntegrationMode = IntegrationMode.PAPER
    
    # Trading parameters
    symbols: List[str] = ["BTCUSDT", "EURUSD"]
    initial_capital: float = 100000.0
    
    # Risk limits
    max_risk_per_trade: float = 2.0   # 2% max per trade
    max_daily_loss: float = 5.0        # 5% daily loss limit
    max_drawdown: float = 20.0         # 20% max drawdown
    max_positions: int = 10            # Max concurrent positions
    
    # System settings
    enable_ai: bool = True             # Enable AI/ML systems
    enable_quantum: bool = False       # Enable quantum computing
    enable_blockchain: bool = False    # Enable blockchain/DeFi
    enable_defi: bool = False          # Enable DeFi features
    enable_sentiment: bool = True      # Enable sentiment analysis
    enable_alternative_data: bool = True  # Enable alt data
```

### Integration Modes

| Mode | Description |
|------|-------------|
