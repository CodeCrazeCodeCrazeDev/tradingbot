## 📝 USAGE EXAMPLES

### Using MLOps
```python
from trading_bot import ModelRegistry, ExperimentTracker

# Register a model
registry = ModelRegistry()
registry.register_model(
    model=my_model,
    model_id="predictor",
    version="v1.0",
    model_type="transformer",
    metrics={"accuracy": 0.85}
)

# Track experiments
tracker = ExperimentTracker()
exp_id = tracker.create_experiment(
    name="Strategy Test",
    description="Testing new entry rules",
    parameters={"stop_loss": 0.02}
)
tracker.log_metrics(exp_id, {"win_rate": 0.65})
```

### Using AI Forecasting
```python
from trading_bot import TemporalFusionTransformer, ForecastEnsemble

# Single model
tft = TemporalFusionTransformer()
tft.fit(X_train, y_train)
predictions = tft.predict(X_test, horizon=24)

# Ensemble
ensemble = ForecastEnsemble()
ensemble.fit(X_train, y_train)
ensemble_predictions = ensemble.predict(X_test)
```
