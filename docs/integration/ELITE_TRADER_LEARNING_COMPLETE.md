## 💻 Usage Examples

### **Quick Start**

```python
from trading_bot.intelligence_core import quick_start_elite_learning

# Initialize the learning system
learning = quick_start_elite_learning()

# Run a learning cycle
cycle_result = learning.run_learning_cycle(
    practice_duration_hours=10.0,
    skills_to_focus=['liquidity_spotting', 'order_flow_reading']
)

# Check mastery report
report = learning.get_learning_report()
print(f"Overall Score: {report['mastery']['overall_score']:.1f}/100")
print(f"Skills Mastered: {report['skills_mastered']}/{report['mastery']['total_skills']}")
```

### **Approve and Implement Improvements**

```python
# Get pending proposals
pending_proposals = [
    p.proposal_id for p in learning.improvement_engine.proposals.values()
    if p.status.value == 'pending_approval'
]

# Human approves proposals
results = learning.approve_and_implement_improvements(
    proposal_ids=pending_proposals[:5],  # Approve top 5
    approved_by="Human Trader"
)

print(f"Implemented: {len(results['implemented'])}")
print(f"Verified: {len(results['verified'])}")
```

### **Simulate Elite Trader Journey**

```python
# Simulate complete learning journey
journey = learning.simulate_elite_trader_journey(

print(f"Final Score: {final_report['mastery']['overall_score']:.1f}/100")
```

### **Check Individual Skill Status**

