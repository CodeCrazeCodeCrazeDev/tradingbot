### 1. Install Dependencies
```bash
pip install -r requirements_aamis_v3.txt
```

### 2. Run Demo
```bash
# Option A: Interactive launcher
RUN_AAMIS_V3.bat

# Option B: Direct execution
python examples/aamis_v3_demo.py
```

### 3. Use in Code
```python
from trading_bot.aamis_v3 import AAMISMasterOrchestrator
import asyncio

async def main():
    aamis = AAMISMasterOrchestrator()
    
    market_data = {
        'macro': {'gdp_growth': 2.5, 'cpi': 3.0},
        'microstructure': {'price': 4500, 'vpoc': 4490},
        'sentiment': {'vix': 16},
        # ... more data
    }
    
    report = await aamis.analyze_market(market_data)
    print(report.executive_summary)

asyncio.run(main())
```
