### Quick Start

```python
from trading_bot.brain.adaptive_integration import AdaptiveIntegrationSystem

# Initialize system

# Process market data
result = adaptive_system.process(market_data, additional_inputs)

# Result contains:
# - decision: BUY/SELL/HOLD
# - confidence: 0.0-1.0
# - market_condition: NORMAL/VOLATILE/EXTREME/TRENDING/RANGING/TRANSITIONING
# - integration_mode: FULL_TIER/FAST_TRACK/EMERGENCY/TREND_FOCUSED/MEAN_REVERSION/ADAPTIVE
# - explanation: Detailed reasoning
# - processing_time: Execution time
```

### With Offline RL

```python
from trading_bot.ml.offline_rl import create_alphaalgo_system
from trading_bot.brain.adaptive_integration import AdaptiveIntegrationSystem

# Create systems
adaptive_system = AdaptiveIntegrationSystem()
alphaalgo_system = create_alphaalgo_system(
    state_dim=50,
    action_dim=3,
    config={
        'training_interval_hours': 24,
        'auto_deploy': True,
        'auto_rollback': True
    }
)

# Start autonomous learning
alphaalgo_system.start()

# Use adaptive integration for decisions
result = adaptive_system.process(market_data, {})

# System will:
# 1. Detect market condition
# 2. Select optimal mode
# 3. Process through tiers
# 4. Return decision
# 5. Learn from outcome
# 6. Improve over time
```
