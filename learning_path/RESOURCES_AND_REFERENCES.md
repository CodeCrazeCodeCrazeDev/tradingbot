## 📰 RESEARCH PAPERS (MUST READ)

### **Foundational Papers**

1. **"The Cross-Section of Expected Stock Returns" (Fama & French, 1992)**
   - Factor models foundation
   - Value and size effects

2. **"A Five-Factor Asset Pricing Model" (Fama & French, 2015)**
   - Modern factor framework
   - Profitability and investment factors

3. **"Momentum Strategies" (Jegadeesh & Titman, 1993)**
   - Momentum effect discovery
   - Trading strategy implications

### **Machine Learning in Finance**

4. **"Deep Learning for Finance" (Heaton et al., 2017)**
   - DL applications in trading
   - Architecture recommendations

5. **"Financial Applications of Machine Learning" (Dixon et al., 2020)**
   - Comprehensive ML survey
   - Best practices

6. **"The Probability of Backtest Overfitting" (Bailey et al., 2015)**
   - Avoiding false discoveries
   - Multiple testing corrections

### **Risk Management**

7. **"The Kelly Criterion in Blackjack, Sports Betting, and the Stock Market" (Thorp, 2008)**
   - Optimal position sizing
   - Risk management

8. **"Risk Parity Portfolios" (Maillard et al., 2010)**
   - Alternative portfolio construction
   - Risk-based allocation

## 🔬 RESEARCH METHODOLOGY

### **Paper Reading Process**

1. **First Pass (10 min)**:
   - Read abstract
   - Look at figures
   - Read conclusion

2. **Second Pass (1 hour)**:
   - Read introduction
   - Understand methodology
   - Note key equations

3. **Third Pass (3 hours)**:
   - Implement code
   - Reproduce results
   - Test variations

### **Strategy Development Process**

1. **Hypothesis** (1 day)
   - Research idea
   - Literature review
   - Initial testing

2. **Implementation** (3 days)
   - Code strategy
   - Write tests
   - Debug issues

3. **Backtesting** (2 days)
   - Historical testing
   - Walk-forward validation
   - Robustness checks

4. **Optimization** (2 days)
   - Parameter tuning
   - Feature selection
   - Risk adjustment

5. **Paper Trading** (1 week)
   - Live testing
   - Performance monitoring
   - Issue resolution

6. **Production** (Ongoing)
   - Live deployment
   - Continuous monitoring
   - Regular updates

### **Key Formulas**:

**Sharpe Ratio**:
```
Sharpe = (Return - RiskFreeRate) / Volatility
```

**Kelly Criterion**:
```
Kelly% = (WinRate * AvgWin - LossRate * AvgLoss) / AvgWin
```

**Maximum Drawdown**:
```
MDD = (Trough - Peak) / Peak
```
